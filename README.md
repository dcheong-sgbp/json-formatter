# JSON Formatter & Validator

A JSON formatter and validator: paste JSON to pretty-print it, minify it, and check it is valid — with the exact error if it is not. Zero dependencies — works in plain HTML, React, Vue, Svelte or Astro.

**▶ [Live demo](https://sgbp.tech/tools/json-formatter)**

```html
<script src="json-formatter.js"></script>
<json-formatter></json-formatter>
```

## What it does

The tool parses your JSON, reports the precise syntax error if it is invalid, and otherwise either indents it for reading or minifies it to a single line. Your keys, values and structure never change — only the whitespace.

## Install

```bash
npm install @sgbp/json-formatter
```

or copy `json-formatter.js` into your project.
