# JSON Formatter & Validator

A JSON formatter and validator: paste JSON to pretty-print it, minify it, and check it is valid — with the exact error if it is not. A free, zero-dependency Web Component you can drop into any site.

```html
<script src="json-formatter.js"></script>
<json-formatter></json-formatter>
```

## What it does

The tool parses your JSON, reports the precise syntax error if it is invalid, and otherwise either indents it for reading or minifies it to a single line. Your keys, values and structure never change — only the whitespace.

## When you'd use it

Debugging an API payload, cleaning up a config file, or shrinking JSON for production.

See the [Usage & API guide](usage.md) for framework examples, or the [guide](guide.md) for tips.
